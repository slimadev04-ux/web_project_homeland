# Tripleten web_project_homeland

# De Pátria para Pátria

## Sobre o projeto: Este é o meu projeto desenvolvido durante a Sprint 5 do curso de Desenvolvimento Web da TripleTen. O objetivo foi criar uma página de destino (landing page) totalmente responsiva, aplicando conceitos avançados de CSS e organização de arquivos. Neste projeto, traduzi um design do Figma para um código funcional aplicando conceitos avançados de CSS Responsivo e Metodologia BEM. Aprendi na prática a converter medidas fixas, como o recuo de 148px da seção Intro, em layouts fluidos que se adaptam perfeitamente a diferentes tamanhos de tela. Utilizei o Flexbox para alternar dinamicamente o alinhamento do conteúdo, garantindo que o texto e as imagens permaneçam organizados tanto em resoluções de desktop quanto em dispositivos móveis. Além disso, implementei boas práticas de tipografia com line-height decimal e imagens elásticas com largura relativa, o que resultou em uma interface moderna, acessível e de fácil manutenção.

## Tecnologias: HTML5/CSS3/Flexbox/CSS Grid/Design responsivo/Metodologia BEM.

## Funcionalidades: Layout Adaptável: Interface totalmente responsiva que se ajusta automaticamente para resoluções de Desktop, Tablet e Mobile. Fidelidade ao Design: Implementação técnica precisa seguindo as medidas e recuos do Figma, como o alinhamento de 148px da seção Intro. Imagens Fluídas: Configuração de mídia que permite o redimensionamento automático das imagens sem distorção.Tipografia Otimizada: Uso de espaçamento entre linhas decimal (line-height) para melhor legibilidade e flexibilidade. Arquitetura BEM: Código CSS modular e organizado por blocos independentes, facilitando a manutenção.
